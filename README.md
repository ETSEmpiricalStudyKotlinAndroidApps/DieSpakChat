# Dev-Skill
1. `Hilt` and `Dagger2` for `DI`
2. `Room` and `EncryptedSharedPreferences` for `Storage`
3. `MVVM` pattern for `Clean Architecture`
4. `Kotlin` 100%
5. `Coroutines` and `RxKotlin` for `Asynchronous`
6. `OkHttp3` and `Retrofit2` for `Networking`
